📚 Book Store App

A MERN Stack (MongoDB, Express, React, Node.js) based web application to manage and explore books.
This project demonstrates CRUD operations, REST APIs, and a modern frontend with React.

🚀 Features

📖 Add, update, delete, and view books

🔍 Search and filter functionality

🎨 Responsive UI with modern design

🌐 RESTful API integration

💾 MongoDB database for persistent storage

🛠 Tech Stack

Frontend: React.js, Tailwind CSS / Bootstrap (mention jo tumne use kiya hai)

Backend: Node.js, Express.js

Database: MongoDB (Mongoose ORM)

Other Tools: Axios, JWT (if authentication hai), dotenv

⚡ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/bookStoreApp.git
cd bookStoreApp

2️⃣ Setup Backend
cd backend
npm install
npm start


👉 Create a .env file in backend folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string

3️⃣ Setup Frontend
cd frontend
npm install
npm start
